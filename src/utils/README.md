## `src/utils/`

アプリ全体で再利用するユーティリティ群です。

### 主な構成

- `colors.js`, `colors/*`: 色パレット・色変換・表示用色の導出
- `noise.js`, `rng.js`: ノイズ/乱数（地形生成の再現性に関わる）
- `paramsDefaults.js`: パラメータのデフォルト・時代（era）定義
- `terrain/*`: 地形生成ロジックの部品（湖/高地/氷河/分類/出力など）
- `pathfinding/*`: BFS/Dijkstra 等の補助（距離マップ）

### 注意

- `terrain/*` は相互依存が多く、挙動互換が重要なので、変更は **小さく・局所的に** 行うのが安全です。
