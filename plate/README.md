# plate

> この `plate/` ディレクトリは旧構成（重複）です。  
> 現在の開発・ビルド対象は **リポジトリ直下**（`c:\\data\\plate\\`）のプロジェクトです。
>
> - 開発: リポジトリ直下で `npm run serve`
> - ビルド: リポジトリ直下で `npm run build`
> - Lint: リポジトリ直下で `npm run lint`

## Project setup
```
npm install
```
